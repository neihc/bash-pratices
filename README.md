# Neihc

## build_proto
### Installation
**Step 1**. Run:
```
[ -d bin  ] || mkdir bin && curl https://raw.githubusercontent.com/neihc/bash-pratices/master/build_proto > bin/build_proto
```

**Step 2**. Add this line to `~/.zshrc` or `~/.bashrc`
```
export PATH="$HOME/bin:$PATH"
```

### Usage (run inside project)
```
build_proto -h
```
